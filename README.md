# DGS AI First

Repositorio de exercicios praticos sobre IA generativa engenharia de prompt engenharia de contexto e RAG.

## Sobre o projeto

Este repositorio reune os entregaveis dos exercicios do cenario-ancora da NovaTech.
A NovaTech e uma empresa de logistica que deseja um assistente de IA para responder duvidas
da equipe de atendimento com base na documentacao oficial.

## Estrutura

```
Prática 1/
    exercicio-fase-1-entendimento.md   enunciado e criterios de avaliacao
    documentos/                        documentos oficiais da NovaTech
        FAQ-atendimento.md
        POL-001-politica-devolucao.md
        PROC-042-frete-especial-v1.md
        PROC-042-v2-frete-especial-revisado.md
        SLA-2024-tabela-sla-clientes.md
        anexo-a-documentacao-simulada-novatech.md
        anexo-b-chunks-referencia-rag.md
    exercicio-1.1/                     viabilidade tecnica (Desenvolvedor 1.1)
    exercicio-1.2/                     prototipacao de prompt (Desenvolvedor 1.2)
        tarefa-1-system-prompt-v1.md
        tarefa-2-mapeamento-contexto.md
        tarefa-3-testes-rodada-1.md
        tarefa-4-analise-critica.md
        tarefa-5-system-prompt-v2-rodada-2.md
```

## Como usar

1. Leia o enunciado em `Prática 1/exercicio-fase-1-entendimento.md`.
2. Consulte os anexos na mesma pasta.
3. Acompanhe os entregaveis em `Prática 1/exercicio-1.1/` e `Prática 1/exercicio-1.2/`.
